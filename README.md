# YouTube Data Analytics Dashboard

## Overview

A Flask-based Data Analytics Dashboard that allows users to upload YouTube datasets and generate insights, visualizations, and machine learning predictions.

## Features

* CSV File Upload
* Data Cleaning
* Engagement Rate Analysis
* Viral Video Detection
* Correlation Analysis
* Dashboard Metrics
* Machine Learning Prediction

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Machine Learning

Linear Regression model used to predict video views based on:

* Likes
* Comment Count
* Engagement Rate

## Project Objective

To analyze YouTube video performance and provide insights into engagement, views, categories, and viral content trends.
